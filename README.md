# Scratch 3 Projects

This repository contains my personal Scratch 3 projects.  
They include animations, games, and interactive stories created for fun and practice.

---

## 📘 Contents
- **Animations**
  - Character movements
  - Background transitions
- **Games**
  - Simple click‑based games
  - Maze challenges
- **Interactive Stories**
  - Dialogues and branching paths
  - Creative storytelling with sprites

---

## 🚀 How to Run
1. Open [Scratch 3] (https://scratch.mit.edu).  
2. Download any `.sb3` file from this repo.  
3. Upload it into Scratch 3 and click the green flag to run.
